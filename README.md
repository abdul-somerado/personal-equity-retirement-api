# Personal Equity & Retirement API

A Spring Boot REST API for managing personal investment portfolios and retirement accounts.

## Features
- User authentication (JWT)
- Portfolio management
- Buy/Sell asset tracking
- Retirement contributions & withdrawals
- Transaction history

## Tech Stack
- Java 11
- Spring Boot
- Spring Security
- MySQL
- Docker

## Run Locally
1. Clone repo
2. Configure DB in application.yml
3. Run:
   mvn spring-boot:run

## API Docs
